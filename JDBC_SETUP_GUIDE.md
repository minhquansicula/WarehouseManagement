# JDBC Driver Setup Guide

## Overview

Ứng dụng Warehouse Management sử dụng Apache Struts 1.3 và JDBC để kết nối database. Hiện tại cấu hình mặc định dùng Oracle Database. Hướng dẫn dưới đây giúp bạn setup JDBC driver và cấu hình kết nối.

---

## 1. Oracle Database Setup (Khuyến nghị)

### 1.1 Tải Oracle JDBC Driver

**Option A: Tải từ Maven Central (Dễ nhất)**

```
https://repo1.maven.org/maven2/com/oracle/database/jdbc/ojdbc11/23.3.0.23.09/ojdbc11-23.3.0.23.09.jar
```

**Option B: Tải từ Oracle Website**

- Truy cập: https://www.oracle.com/database/technologies/appdev/jdbc.html
- Chọn phiên bản phù hợp với Java version:
  - Java 8: ojdbc8.jar
  - Java 11+: ojdbc11.jar
  - Java 21+: ojdbc23.jar

### 1.2 Đặt JAR vào Project

1. Tạo thư mục nếu chưa có: `WebContent/WEB-INF/lib/`
2. Copy file `ojdbc*.jar` vào thư mục trên
3. Rebuild project:

```bash
cd E:\PRACTICE\WarehouseManagment
ant clean compile war
```

### 1.3 Cấu hình Kết nối Database

Mở file: `src/java/util/DBConnectionManager.java`

Sửa các giá trị theo database của bạn:

```java
private static final String URL = "jdbc:oracle:thin:@localhost:1521:XE";  // địa chỉ Oracle
private static final String USER = "system";                              // username
private static final String PASS = "12345";                               // password
```

**Ví dụ cho các cơ sở dữ liệu khác:**

- Oracle XE (local): `jdbc:oracle:thin:@localhost:1521:XE`
- Oracle Cloud: `jdbc:oracle:thin:@your_host:1521:your_service_name`
- MySQL: `jdbc:mysql://localhost:3306/warehouse`
- H2 (embedded): `jdbc:h2:mem:warehouse`

### 1.4 Tạo Bảng Users trong Database

1. Kết nối vào Oracle (SQL\*Plus hoặc SQL Developer)
2. Chạy script SQL từ file: `DATABASE_SETUP.sql`

```sql
-- Chạy toàn bộ script từ DATABASE_SETUP.sql
-- Hoặc chạy lệnh này (Oracle):
CREATE TABLE users (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    username VARCHAR2(100) UNIQUE NOT NULL,
    password VARCHAR2(255) NOT NULL,
    role VARCHAR2(50) DEFAULT 'USER',
    created_at TIMESTAMP DEFAULT SYSTIMESTAMP
);

INSERT INTO users (username, password, role) VALUES ('admin', 'admin', 'ADMIN');
INSERT INTO users (username, password, role) VALUES ('user1', 'user123', 'USER');

COMMIT;
```

## 2. Cấu Hình Khác (MySQL / H2)

### 2.1 Chuyển sang MySQL

1. **Tải MySQL JDBC Driver:**

```
https://repo1.maven.org/maven2/mysql/mysql-connector-java/8.0.33/mysql-connector-java-8.0.33.jar
```

2. **Copy vào `WebContent/WEB-INF/lib/`**

3. **Sửa `DBConnectionManager.java`:**

```java
private static final String URL = "jdbc:mysql://localhost:3306/warehouse";
private static final String USER = "root";
private static final String PASS = "your_password";
private static final String DRIVER = "com.mysql.cj.jdbc.Driver";
```

4. **Chạy SQL (MySQL):**

```sql
CREATE DATABASE warehouse;
USE warehouse;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(100) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    role VARCHAR(50) DEFAULT 'USER',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

INSERT INTO users (username, password, role) VALUES ('admin', 'admin', 'ADMIN');
INSERT INTO users (username, password, role) VALUES ('user1', 'user123', 'USER');
```

### 2.2 Chuyển sang H2 (In-Memory - Cho testing)

1. **Tải H2 JDBC Driver:**

```
https://repo1.maven.org/maven2/com/h2database/h2/2.1.214/h2-2.1.214.jar
```

2. **Copy vào `WebContent/WEB-INF/lib/`**

3. **Sửa `DBConnectionManager.java`:**

```java
private static final String URL = "jdbc:h2:mem:warehouse;DB_CLOSE_DELAY=-1;DB_CLOSE_ON_EXIT=FALSE";
private static final String USER = "sa";
private static final String PASS = "";
private static final String DRIVER = "org.h2.Driver";
```

4. **Thêm init script (tạo table tự động) - tuỳ chọn:**

```
jdbc:h2:mem:warehouse;INIT=CREATE TABLE users (id INT PRIMARY KEY AUTO_INCREMENT, username VARCHAR(100) UNIQUE NOT NULL, password VARCHAR(255), role VARCHAR(50));
```

## 3. Build & Deploy

### 3.1 Rebuild Project

```bash
cd E:\PRACTICE\WarehouseManagment
ant clean compile war
```

### 3.2 Deploy WAR

- **Nếu dùng Tomcat Manager**: Upload `dist/WarehouseManagement.war` vào webapps
- **Nếu dùng Tomcat exploded**: Copy `dist/WarehouseManagement/` vào `$CATALINA_HOME/webapps/WarehouseManagement/`

```bash
$CATALINA_HOME/bin/startup.bat  (Windows)
$CATALINA_HOME/bin/startup.sh   (Linux/Mac)
```

## 4. Test Login

1. Truy cập: `http://localhost:8080/WarehouseManagement/jsp/login.jsp`
2. Đăng nhập bằng:
   - Username: `admin`
   - Password: `admin`
3. Nếu thành công, chuyển hướng tới dashboard
4. Nếu có lỗi, kiểm tra:
   - Tomcat logs: `$CATALINA_HOME/logs/catalina.out`
   - JDBC driver có trong `WEB-INF/lib` không
   - Database connection parameters đúng không
   - Users table đã được tạo không

## 5. Troubleshooting

| Lỗi                                                       | Nguyên nhân                                     | Giải pháp                                                           |
| --------------------------------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------- |
| `ClassNotFoundException: oracle.jdbc.driver.OracleDriver` | JDBC driver không tìm được                      | Copy ojdbc\*.jar vào `WEB-INF/lib/`, rebuild                        |
| `SQLException: ORA-01017: invalid username/password`      | Username/password sai hoặc users table chưa tạo | Kiểm tra DB, chạy `DATABASE_SETUP.sql`                              |
| `SQLException: connection refused`                        | Database server không chạy hoặc host/port sai   | Kiểm tra DB connection params, khởi động database server            |
| CSS/JS file 404                                           | Static resource path sai                        | Đã sửa xong - dùng `<%=request.getContextPath()%>` cho tất cả asset |
| Session không lưu `currentUser`                           | Login thất bại vô hình                          | Kiểm tra browser console, Tomcat logs                               |
